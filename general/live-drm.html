實時數字版權管理

在本主題中，您將學習如何為即時串流啟用內容保護。

啟用即時內容的 DRM

注意：

默認情況下，DRM 應用於輸出，如下所示：

  • 必須先啟用您的真實賬戶 由Brightcove支持以使用DRM。

  • 若要發佈已啟用 DRM 的事件，請使用下列任一項作業：

    • 已啟用 DRM 的布萊特灣播放器 v6.37.1 或更新版本
    • 適用於 iOS 版本 6.5.0 或更新版本的布萊特灣原生 SDK

    • 適用於安卓版 6.11.0 或更高版本的布萊特灣原生 SDK

  • 如果您想覆蓋 DRM 默認設置，您可以通過 API 進行操作，或者請聯繫 Brightcove 支持在即時平台中更改它們。
    • 平均播放：應用於 HLS 輸出
    • 播放準備：套用至 DASH 輸出
    • Widevine：應用於HLS輸出[1-1]

筆記

  • [1-1] Widevine 可以應用於 DASH 輸出而不是 HLS。要使用 DASH 使用 Widevine 默認值需要帳戶級別覆蓋。如果您想要將此覆寫套用至您的帳戶，請連絡支援部門。使用實時 API你也可以指定widevine:hls或者widevine:dash對於特定作業，無論您帳戶的默認設置如何。請注意，根據您的帳戶/職位的配置方式，API響應（如下圖所示）將包括widevine:hls或者widevine:dash輸出，但不能同時輸出。

若要為您的即時內容啟用 DRM，請按照下列步驟操作：

  1. 在 [進階選項] > [內容加密] 中。選擇數字版權管理（DRM）從“加密"下拉列表中。
  2. 額外的令牌到期選項將會出現，您可以選擇以秒為單位的時間，最小有效期限為900秒，最大為2592000秒（最後等於30天）。
  3. 在活動資訊面板中，您可以看到公平遊戲和威德維恩的 DRM 網址。
  4. 若要發佈已啟用 DRM 的事件，請按一下左側導覽中的「發佈與嵌入」>「播放器設定」。
    • 在實時播放器選項，選擇啟用DRM的播放器。（要進行實時DRM播放，您需要選擇或創建一個啟用DRM的播放器v6.37.1或更高版本）。
    • 如果沒有可用的DRM播放器，請轉到播放器模塊 >在左側導航中選擇或創建播放器>播放。選擇“啟用DRM"選項。(即時 DRM 播放的播放程式必須為 v6.37.1 或更高版本)。
  5. 回到直播模組中，您將能夠選擇啟用 DRM 的播放器並發佈 DRM 事件。
