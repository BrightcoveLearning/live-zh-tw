    檢視布萊特灣系統狀態頁面

    在本主題中，您將檢視 Brightcove 系統狀態頁面，以取得有關 Brightcove 服務和應用程式的更新。

    布萊特灣持續監控所有布賴特灣服務和應用的狀態。如果服務中斷，系統狀態頁面會張貼一則備註。所有定期的維修通知也將在此頁面上張貼。

    若要存取「系統狀態頁面」，請造訪 https://status.brightcove.com 或瀏覽標題，按一下「支援」圖示 ( )，然後按一下「 系統狀態」。

    所有張貼的通知都會包含下列資訊：

    • 事件狀態
      • 運作中 (綠色)
      • 通知 (藍色 — 系統目前正在運作，但受到延遲或間歇性中斷影響)
      • 部分服務中斷 (黃色)
      • 全面服務中斷 (紅色)
    • 元件-服務或受影響區域
    • 地點
      • 亞太地區
      • 美洲
      • 歐美
      • 日本
    • 目前的狀態和狀態。狀態包括：
      • 調查-工程師正在研究這個問題
      • 確定-工程師已經確定了這個問題，並正在尋找糾正
      • 監控-問題解決和工程監控，確保一切健康
      • 已解決-問題已解決

    所有發佈的訊息都以英文顯示，使用的時間為東部標準時間 (GMT-5)。

    檢視系統狀態歷史記錄

    若要檢視「系統狀態」歷史記錄，請按一下頁面底部的「歷史記錄」連結。

    訂閱系統狀態變更

    您可以註冊，以便在系統狀態變更時收到通知。本節的組織方式如下：

    • 訂閱電子郵件
    • 訂閱網頁鉤
    • 管理您的訂閱

    訂閱電子郵件

    若要透過電子郵件接收通知，請按一下訂閱按鈕，輸入您的電子郵件地址，然後按一下「訂閱」

    訂閱網頁鉤

    網絡鉤是用 HTTP POST 製作的定義回調。您可以指定一個 URL，當有狀態更新時，回調將發布 JSON 對象。

    若要註冊以便在系統狀態變更時收到通知，請選取「訂閱」按鈕。然後輸入一個 URL 以接收 JSON 承載以及您的電子郵件地址。

    管理您的訂閱

    訂閱之後，您會收到一封電子郵件，其中包含管理訂閱頁面的連結，讓您管理訂閱。您可以選擇您感興趣接收通知的元件/產品/地區。進行選取，然後按一下 [ 儲存訂閱 ]。

    若要取消訂閱，請按一下「管理訂閱」頁面底部的「取消訂閱」連結。
