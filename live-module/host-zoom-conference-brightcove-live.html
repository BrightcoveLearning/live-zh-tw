簡介

變焦視頻通訊是一家技術公司，提供基於雲的對等軟件平台，用於電話會議，遠程通勤和遠程教育。若要接觸更多觀眾，您可以利用 Brightcove Live 串流您的會議，讓其他人觀看。

步驟

請依照下列步驟舉辦會議或網路研討會，參加 Brightcove 直播：

  1. 會議

    對於 Zoom 會議，請執行下列動作：

    1. 導航到您的Zoom Meeting 設定值並滾動到會議中（高級部分。
    2. 啟用允許直播會議單擊撥動開關選擇選項。如果您沒有看到此選項，請洽詢您的縮放帳戶管理員，了解如何將此功能新增至您的帳戶。

    3. 選擇定制直播服務，還可以在框中輸入文字註釋。

    網路研討會

    對於「縮放」網路研討會，請執行下列動作：

    1. 導航到您的“縮放網絡研討會設置"，然後滾動到在網絡研討會設置中部分。
    2. 啟用允許主持人直播他們的網絡研討會選項。

    3. 選擇定制流媒體服務，然後保存。

  2. 在Video Cloud Studio中，創建一個使用媒體模塊的現場直播。

    如果您想在完成後保存廣播，請務必檢查完成後將事件轉換為視頻資產複選框！

  3. 當您建立即時事件時，您將可以存取串流 URL 和串流金鑰。

    • 串流 URL-串流端點 URL (您可以在 Brightcove 即時控制室找到端點 URL
    • 流鍵-流名稱（在Brightcove中，流名稱始終為“活"）
  4. 在您的網站/內部網路頁面中發佈並嵌入此 Live 活動的播放程式，並記下網頁 URL。
  5. 會議

    對於Zoom會議，請導航到您的預定會議列表並選擇您要流式傳輸的會議。

    網路研討會

    對於Zoom網絡研討會，請返回網絡研討會屏幕，並安排一個網絡研討會。選擇即時串流標籤。

  6. 滾動到即時串流部分
  7. 選擇配置直播設置關聯。您將在這裡輸入串流網址、串流索引鍵以及代管播放器的頁面的 URL。
  8. 在您的Zoom通話中，選擇更多的按鈕，然後選擇直播自定義直播服務選項。這將開始直播。
