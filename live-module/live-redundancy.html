啟用即時備援

注意事項：

  • 每個帳戶都有一個 SEP（靜態入口點）限制為 3，除了我們-west-2，這將有 10 的限制。如果您需要增加，請聯繫布萊特灣支援部門。
  • 如果您將冗餘與SSAI一起使用，則必須同時將廣告插入組中的所有作業中。

即時備援功能可透過在 Brightcove 直播平台中新增兩個或更多串流至您的活動。

首先建立即時活動。（有關如何創建直播活動的更多信息，請點擊這裡）。

  1. 轉到流選項，這些選項將允許您命名和選擇流的區域。點擊（+）添加備份流按鈕。

    如果沒有看到「新增備份串流」按鈕，表示您的帳戶尚未啟用備份串流。

    筆記：請聯繫 Brightcove 支持以在您的帳戶中啟用此選項。
  2. 通過點擊添加備份流按鈕，您將為實時視頻添加備份（冗餘）流。您可以加入帳戶上限所支援的數量。
    您可能有多餘的組計入您的帳戶限制。一旦具有冗餘流的事件完成或被取消，它就不再可見在即時模塊中。不過，這類事件可以透過即時 API 列出和刪除。看實時API：冗餘組。
  3. 命名您的主要和備份串流，然後選擇最接近編碼器所在的地區。默認情況下，主流地區設置為俄勒岡州（美國西部），如果您是從其他大洲來的話，將會產生不好的結果。

    這參考編號是內部使用的可選標識符，並且必須是唯一的（最多150個字符）。

    繼續按照平常的方式設定直播活動。有關如何創建直播活動的更多信息，請點擊這裡。

  4. 或者，您也可以從控制室視窗加入新的串流：
  5. 新增備份串流資訊和地區。
  6. 在控制室中，您將可以看到終點和流名稱每個流的詳細信息。
  7. 您可以在控制室。

注意：如果您建立單串流事件，並在 Control Room 中新增備援資料流，就必須重新發佈播放器內嵌程式碼，才能從冗餘中受益。

裁切

  1. 只要從下拉式功能表中選取想要的串流，即可在每個串流中使用 [視訊剪輯]。有關創建剪輯的信息，請參見使用實時模塊創建視頻剪輯。
  2. 第二個資料流選取。

監視

按一下預覽播放器下方的監控索引標籤以檢視監控資訊。只要從下拉式功能表中選取想要的串流，即可針對每個串流進行監控。

在事件進行期間，事件結束後最多 1 小時內都可以使用監控資訊。

刪除冗餘組

創建冗餘組後：

  1. 點擊取消流來自小組的一項工作。
  2. 點擊停止此流來自小組的第二份工作。
  3. 前往管理活動頁面並選擇完全的標籤。
  4. 冗餘事件應該在閒置的狀態。
  5. 點擊關閉刪除冗餘事件。

限制

  • 實時冗餘不能與受DRM保護的直播流一起使用。
