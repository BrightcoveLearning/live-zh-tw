將 Zoom 會議集成到 Brightcove Live

Brightcove Live Zoom集成使您可以使用Brightcove Live輕鬆廣播Zoom會議。

先決條件

在Zoom Marketplace中

與Brightcove Live的縮放集成由Brightcove Live Connect縮放市場中的應用程序。這是一個帳戶級別的應用程序，僅供授權的所有者或管理員級別的Zoom帳戶使用。請確保您的Zoom帳戶具有支持集成的正確特權。

在Brightcove視頻雲中

在開始集成之前，您必須具有一個自定義元數據字段，以跟踪事件是否正在從Zoom流式傳輸。這將跟踪會話之間的集成狀態，即當前是否將某個事件鏈接到Zoom會議。有關如何添加自定義元數據字段的具體詳細信息，請查看以下文件。

  1. 去Video Cloud Studio > 行政 > 視頻場 > 添加自定義字段
    2. 。
  2. 寫一個顯示名稱用於您的自定義元數據字段。
  3. 為了內部名稱字段，你應該寫： bc_meeting_id
    不要將任何外部縮放 ID 或數據添加到自定義字段中。
  4. 寫一個描述用於您的自定義元數據字段。

最佳實務

為了獲得最佳效果，請按照以下說明進行操作。

  1. 使用開始和停止控制室而不是主機Zoom播放器中的廣播按鈕。
  2. 避免在單個帳戶下同時流式傳輸多個Zoom會議。

創建一個縮放會議

  1. 首先，訪問您的縮放帳戶，然後創建一個縮放會議。有關如何創建Zoom會議的信息，請按照放大文檔。
  2. 在縮放中，轉到設定值 > 會議（高級） > 直播會議。
  3. 確保您的Zoom帳戶設置具有定制直播服務已啟用。

將Zoom帳戶與Brightcove Live集成

要將Zoom帳戶與Brightcove Live集成，請按照下列步驟操作：

  1. 登入視訊雲端工作室。
  2. 點擊行政導航標題中的按鈕。
  3. 選擇整合方式下拉菜單中的選項。
  4. 點擊添加集成按鈕將一個或多個Zoom帳戶鏈接到Brightcove Video Cloud。
  5. 將會出現一個彈出窗口，要求您訪問您的Zoom帳戶。點擊授權按鈕。
  6. 提供您的Zoom憑據以集成該帳戶。

將Zoom集成添加到Brightcove Live事件

要將Zoom集成添加到Brightcove Live事件，請按照下列步驟操作：

  1. 導航到實時模塊在視頻雲中。
  2. 點擊建立活動直播按鈕管理活動頁。有關如何創建直播活動的完整信息，請查看使用實時模塊創建和管理實時事件文檔。
  3. 像往常一樣填寫直播活動的詳細信息，直到您到達整合方式部分。
  4. 點擊添加集成按鈕。
  5. 您將能夠看到正在進行的和預定的Zoom會議的列表。
  6. 點擊下拉菜單。
  7. 選擇會議。在此示例中，我選擇了測試文件會議。
    一次只能向一個事件添加一個集成。要清除集成，請單擊刪除集成按鈕。如果您想擁有多個帳戶，您可以在行政>集成控制板。
  8. 準備好創建直播活動後，請點擊建立活動按鈕。
  9. 在裡面控制室，將有一個選項卡式選項可用於通過Zoom傳輸會議。如果您不想使用集成源，請切換回左上方的Ingest URL標籤。
  10. 如果您最近創建了Zoom會議，請嘗試使用再次與Zoom同步同步按鈕。
    如果您的會議在同步後仍未顯示，請嘗試驗證您的會議是否已預定在 Zoom 中，並且您安排的會議少於 30 次。
  11. 前往您的變焦會議帳戶。
  12. 點擊藍色開始按鈕開始您的會議。
    由於 Zoom 限制，在開始向 Brightcove Live 廣播 Zoom 會議之前，必須先開始 Zoom 會議。
  13. 回到Brightcove Live 控制室， 點擊開始廣播按鈕。
  14. 完成最後一步後，您將能夠在控制室 .
    • 在縮放會議期間的任何時間，您都可以停止並重新開始廣播。但是，如果您未積極廣播事件，則如果間隔超過事件重新連接時間，則事件可能會超時。
    • 通過Brightcove播放的變焦廣播將延遲大約20秒。
    • 如果Zoom會議在20秒後仍未開始流式傳輸，則可以單擊停止廣播然後開始廣播再次。
    • 由於Zoom流是第三方服務，因此可能不支持某些高級實時功能，包括SSAI。

卸載Zoom Integration

從Video Cloud刪除集成

要在Brightcove中從您的帳戶中刪除Zoom集成，請執行以下操作：

  1. 登錄到Video Cloud Studio。
  2. 點擊行政導航標題中的按鈕。
  3. 選擇整合方式下拉菜單中的選項。
  4. 點擊垃圾圖標。
  5. 當要求確認時，請單擊確認按鈕。

從Zoom帳戶中刪除Brightcove Live Connect

您可以完全刪除Brightcove Live Connect通過連接到Zoom，從您的Zoom帳戶中訪問該應用：

  1. 登錄到您的Zoom帳戶並導航到縮放應用市場。
  2. 點擊管理>已安裝的應用或搜索布萊頓灣應用程序。
  3. 點擊布萊頓灣應用程序。
  4. 點擊卸載。
