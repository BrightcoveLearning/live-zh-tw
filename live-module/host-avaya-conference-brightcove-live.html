簡介

Avaya Rooms系統是一個視頻會議應用程序，它使您可以在線與其他人見面。若要接觸更多觀眾，您可以利用 Brightcove Live 串流您的會議，讓其他人觀看。

步驟

請依照下列步驟舉辦會議至布萊特灣直播：

  1. 在Video Cloud Studio中，創建一個使用實時模塊的實時事件。
    如果您想在完成後保存廣播，請務必檢查完成後將事件轉換為視頻資產複選框！

  2. 創建實況事件後，請從控制室控制板。

    • 流式RTMP URL- 流端點URL（您可以在Brightcove Live Control Room中找到端點URL
    • 流密鑰- 流名稱（在Brightcove中，流名稱始終為“活"）
  3. 在您的網站/內部網路頁面中發佈並嵌入此 Live 活動的播放程式，並記下網頁 URL。

設定串流階段作業

使用串流 RTMP URL 和串流金鑰，您必須在 Avaya 轉碼器單元上進行設定。

  1. 在Avaya GUI中，轉到配置>流。

    （可選）在Avaya網站中，轉到管理者設置>實用程序>流。

  2. 插入串流 RTMP 網址。
  3. 插入您的串流金鑰。

啟動/停止串流工作階段

若要啟動或停止串流工作階段，請依照下列步驟執行：

  1. 去主頁>更多>流。
  2. 點擊流媒體按鈕。這將開始直播。
  3. RTMP 串流預設為啟用。要禁用它，請在Avaya平台中轉到配置>高級>實用程序>流。
