    使用即時 RTMP 輸出串流至臉書和 YouTube

    在本主題中，您將學習如何使用即時模組中的 RTMP 輸出，將即時活動串流至 Facebook 和 YouTube。

    簡介

    直播模組可用於直播活動至桌上型電腦和行動裝置。在直播模組中輸入您的活動資訊後，系統會提供一組編碼器設定，讓您插入現場編碼器。一般而言，直播會在布萊特灣播放器內發佈。「即時模組控制室」頁面提供輸入其他 RTMP 輸出 URL 的功能。RTMP 輸出設定可用於將即時串流傳送至社交媒體網站。請注意，RTMP輸出小時數將根據事件小時數計費。 本主題將介紹創建簡單的實時事件的步驟，然後說明如何使用RTMP輸出將實時事件流式傳輸到Facebook和YouTube。

    注意：廣告 (包括伺服器端廣告 (SSAI) 不會串流至 RTMP 輸出。系統會使用無廣告串流。

    建立新事件

    若要建立新的事件，請依照下列步驟執行。

    1. 開啟即時模組。
    2. 在“管理事件”頁面中，單擊建立活動按鈕或單擊建立現場活動要么創建直播頻道在左側導航欄中。「建立即時事件」或「建立即時頻道」頁面隨即開啟。
    3. 輸入活動名稱以及任何必填的自定義字段。
    4. 選取「即時內嵌設定檔」。
    5. 選取 [ 完成時將事件轉換為視訊資產 ]，即可將直播影片的 VOD 版本儲存到您的 Video Cloud 帳戶。
    6. 請點擊高級選項並適當配置高級選項。
    7. 請點擊開始流式傳輸開始直播活動。[控制室] 頁面將會開啟，其中包含即時事件的詳細資料，包括編碼器將使用的串流端點 (RTMP URL) 和串流名稱。
      注意：處理串流需要大約一分鐘的時間。
      注意：根據預設，編碼器必須在建立新事件後的 30 分鐘內啟動。
    8. 從編碼器開始串流，直播就會出現在「控制室」頁面中。

    直播活動開始後，可以將直播流發送到臉書和/或的YouTube。

    注意：確保編碼器在即時串流完成時關閉/停止。在任何編碼器和 Brightcove 之間保持連線開啟，即使沒有傳送任何資料，也可能會導致未預期的連線費用。

    串流至臉書直播

    Facebook Live  是一個 生活 上的視頻流功能 臉書 讓您播放 生活 通過公司頁面或個人資料向受眾群體展示事件。直播視頻將在 直播活動以及 活動已結束。

    注意：有關Facebook直播的完整詳細信息，請查看Facebook文件。

    直播模組開始直播後，請依照下列步驟在 Facebook 中使用直播。

    1. 登入您的臉書帳號。
    2. 創建一個新帖子，然後單擊實況視頻選項。
    3. Facebook 可能會提示您存取您的電腦攝影機和麥克風。本主題使用外接數碼單鏡反光相機進行活動，因此請按一下「連接」。
    4. Facebook將顯示一個服務器網址和流密鑰。這些值將在「即時」模組中使用。
    5. 將伺服器 URL 複製到剪貼簿。
    6. 在「即時」模組中，將伺服器 URL 貼到「控制室」頁面的 RTMP 輸出 URL 欄位中。
    7. 複製流密鑰從Facebook到剪貼板。
    8. 在實時模塊中，粘貼流密鑰在結束時RTMP輸出網址領域。的RTMP輸出網址領域將是Facebook 服務器網址與流密鑰附加，全部作為一個值。
      注意： RTMP URL 目前支援的唯一格式為rtmps://host:port/application/streamName。
    9. 從「選取節目」下拉式清單中選取節目，然後按一下「新增 RTMP 輸出」。
    10. 在 RTMP 區段下方，確認連線成功。
    11. 確認即時直播出現在臉書中。輸入臉書的標題和說明。
    12. 請點擊上線。

    直播將在臉書開始。

    請點擊完結束Facebook直播。事件完成時，請記得停止編碼器。

    串流至 YouTube

    YouTube 提供即時串流影片給觀眾的能力。首次開始直播之前，您必須先啟用 YouTube 頻道。

    注意：啟用 首次直播可能需要24小時。啟用後，您的 流可以立即直播。
    注意：有關YouTube直播的完整詳細信息，請檢查YouTube文檔。

    直播模組開始直播後，請依照下列步驟在 YouTube 中使用直播。

    1. 登入您的 YouTube 帳戶。
    2. 轉到YouTube Studio頁面https://studio.youtube.com。
    3. 在左側導航中，單擊其他功能>立即直播。
    4. YouTube 可能會提示您存取您的電腦攝影機和麥克風。本主題使用外接數碼單鏡反光相機進行活動，因此請按一下「連接」。
    5. YouTube將顯示一個服務器網址和流名稱/密鑰。這些值將在「即時」模組中使用。
    6. 將伺服器 URL 複製到剪貼簿。
    7. 在「即時」模組中，將伺服器 URL 貼到「控制室」頁面的 RTMP 輸出 URL 欄位中。
      注意： RTMP URL 目前支援的唯一格式為rtmps://host:port/application/streamName。
    8. 複製流名稱/密鑰從YouTube到剪貼板。
    9. 在實時模塊中，粘貼流名稱/密鑰在結束時RTMP輸出網址領域。的RTMP輸出網址字段將是YouTube 服務器網址與流名稱/密鑰附加，全部作為一個值。
    10. 從「選取節目」下拉式清單中選取節目，然後按一下「新增 RTMP 輸出」。
    11. 在 RTMP 區段下方，確認連線成功。
    12. 確認直播顯示在 YouTube 上。

    直播將在 YouTube 開始播放。

    請點擊返回YouTube Studio然後停止編碼器以結束直播。串流完成後，系統會自動建立並上傳公開影片，供觀眾稍後觀看。
