簡介

INXPO是一個視頻流媒體平台，用於創建電視風格的事件體驗。若要接觸更多觀眾，您可以利用 Brightcove 直播來串流您的賽事。

步驟

請依照下列步驟舉辦會議至布萊特灣直播：

  1. 在Video Cloud Studio中，創建一個使用實時模塊的實時事件。
    如果您想在完成後保存廣播，請務必檢查完成後將事件轉換為視頻資產複選框！

  2. 創建實況事件後，請從控制室控制板。

    • 流式RTMP URL- 流端點URL（您可以在Brightcove Live Control Room中找到端點URL
    • 流密鑰- 流名稱（在Brightcove中，流名稱始終為“活"）
  3. 在您的網站/內部網路頁面中發佈並嵌入此 Live 活動的播放程式，並記下網頁 URL。

設定串流階段作業

使用串流 RTMP URL 和串流金鑰，您必須在 INXPO 製作者工具上進行設定。

  1. 移至 INXPO 製作人工具，然後按一下 3 個點圖示，即可到達 [輸出詳細資料] 區段。
  2. 按一下鉛筆圖示。
  3. 使用您的 Brightcove 直播活動憑證填寫以下詳細資料。
  4. 在裡面標題字段中，添加Brightcove實時流名稱。
  5. 在裡面編碼配置文件字段，選擇默認的編碼配置文件，該設置定義了編碼器如何壓縮網絡廣播並將其傳遞給與會者。
  6. 在裡面RTMP網址字段，插入Brightcove Streaming Endpoint URL。
  7. 點擊好的按鈕，僅此而已。你就是活下去
