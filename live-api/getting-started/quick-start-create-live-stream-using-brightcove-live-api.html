    快速入門：使用布萊特灣即時 API 建立即時串流

    • 為Live API設置的帳戶

    設置視頻雲視頻

    現在我們有了實時流信息，我們將在Video Cloud中設置一個遠程視頻，以便我們可以在Brightcove Player中播放它。

    在這些步驟中，我們使用CMS API，但如果您願意，也可以在Video Cloud Studio中完成-請參閱下面的替代步驟。

    如果您使用的是Insomnia或Postman，則需要使用客戶端憑據設置OAuth2身份驗證。如果需要幫助，請參閱以下指南之一：

    • 針對 API 請求使用失眠
    • 使用郵遞員進行 API 請求

    1. 首先，您需要通過製作一個新視頻POST請求：

      https://cms.api.brightcove.com/v1/accounts/account_id/videos

    2. 請求主體可以包含幾段元數據（請參閱API參考（有關詳細信息），但唯一需要的是name：

      {
    "name": "some name for your live stream"
}

    3. 響應將包含所有視頻元數據，但重要的字段是id，您將需要添加遠程資產。

    4. 要添加HLS流，您將創建另一個POST請求：

      https://cms.api.brightcove.com/v1/accounts/account_id/videos/video_id/assets/hls_manifest

    5. 這次，請求正文將為：

      {
    "remote_url": "playback_url for your live stream"
}

    使用有線廣播流式傳輸事件

    在本節中，我們將設定並使用 Telestream Wirecast 編碼器來支援即時串流事件。Telestream Wirecast 軟體是一個桌面應用程式，可以從您的相機擷取輸入，並產生可以由 CDN 傳遞的資料流。請注意，有些硬體和其他軟體的編碼解決方案可能更適合用來傳遞您的直播活動。您可以從以下網站下載Wirecast軟件的試用版： Telestream網站。

    注意：繼續進行之前，您應該安裝 Wirecast 軟體，並將相機連接至您的電腦。

    若要設定 Wirecast 的即時事件，請依照下列步驟執行。

    1. 開啟「線上轉播」
    2. 將鼠標懸停在+，點擊捕獲設備然後選擇您的相機。
    3. 點擊箭頭按鈕（），以使相機拍攝到實況鏡頭。
    4. 按一下「資料流」按鈕 ( )。系統將提示您配置輸出設置。
    5. 將 [ 目的地 ] 設定為 RTMP 伺服器。按一下「確定」。
    6. 為了地址， 使用stream_url從Live API返回（第3步）。
    7. 為了流，活著使用。
    8. （可選的）要創建其他輸出流，請單擊加...創建其他輸出流時，您可能希望創建自己的自定義編碼配置文件以控制輸出比特率。確保流名稱反映了新的比特率。
    9. 按一下「確定」以儲存輸出設定。
    10. 按一下「資料流」按鈕 ( )。現場直播應該開始。

    結論

    現在，您可以創建實時流會話了。您需要在編碼器配置中添加以下內容：

    • stream_url：添加stream_url從你的工作中rtmp地址）
    • stream_name：流名稱必須為alive
