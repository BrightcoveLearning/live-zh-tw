簡介

使用「即時」模組建立即時事件時，您必須選取「即時內嵌」設定檔，以決定 Brightcove Live 所建立的轉譯數量和品質。本主題提供每個標準即時內嵌設定檔的詳細資訊。

筆記：
  • 對於透過即時 API 建立的工作，您目前無法指定內嵌設定檔，但必須改為定義輸出特性。
  • 這些設定檔專門用於建立即時事件時。將視訊上傳至視訊雲端時不會使用這些檔案。

您也可以建立自己的自訂即時內嵌設定檔。有關信息，請參閱創建自定義實時攝取配置文件。

文件資料

筆記：
  • 此頁面是使用 API 的即時資料建立；完全載入可能需要一分鐘的時間。
  • 標準配置文件可能不是進行編輯-如果您想要修改的版本，則需要創建一個自定義配置文件。
