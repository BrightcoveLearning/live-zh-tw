需求

  • 您帳戶的實時API密鑰。
  • 您的真實帳戶必須啟用了計劃程序替代。
  • 現有的實時作業“靜態”：真（看靜態入口點作業想要查詢更多的信息）。
請注意，您還可以在實時模塊中安排激活和停用 - 請參閱控制室中實時事件的自動開始/停止詳情

創建一個調度程序

假設我們要安排一個 2 小時的活動，該活動將在2021-04-02T14:00:00+00:00 .

時間戳記必須採用UTC格式，以UNIX時間戳記為單位，以秒為單位。您可以使用https://www.unixtimestamp.com/進行轉換。

請注意，描述的激活時間限制這裡仍然適用於預定的激活，因此我們建議將時間安排在您想要的開始時間之前至少 60 秒。

對於此示例，我們將選擇預期的開始時間之前的5分鐘：

開始：2021-04-02T13:55:00+00:00 = 1617386100 停止：2021-04-02T16：下午四點以上

要實現調度程序，請執行以下操作POST要求：

https://api.bcovlive.io/v1/scheduler/jobstartstop

標頭

Content-Type: application/json
		X-API-KEY: {your API Key}

要求正文

{
			"job_id": "your Job ID",
			"activation": {
				"time_utc": 1617386100
			},
			"deactivation": {
				"time_utc": 1617393600
			},
			"description": "My Wednesday broadcast"
		}

樣品回复

 
{
	"account_id": "1752604059001",
	"activation": {
		"state": "pending",
		"time_utc": 1617386100
	},
	"deactivation": {
		"state": "pending",
		"time_utc": 1617393600
	},
	"description": "My Wednesday broadcast",
	"job_id": "15981d93877e490185e6ebc9af2c79a3",
	"type": "jobstartstop",
	"workflow_id": "d44cf29f4f184757ac0995beb8a0097e"
}

更新調度程序

你可以更新激活和/或停用調度程序的時間，前提是作業處於待辦的狀態 （“狀態”：“待定”）。為此，請發送放請求：

https://api.bcovlive.io/v1/scheduler/jobstartstop/{workflow_id}

標題與郵政請求如上所示。

要求正文

在請求正文對像中，僅包含激活和/或停用領域。

例如，要比計劃提前30分鐘停止活動：

要求正文

{
	"deactivation": {
		"time_utc": 1617391800
	}
}

樣品回复

 
{
	"account_id": "1752604059001",
	"activation": {
		"state": "pending",
		"time_utc": 1617386100
	},
	"deactivation": {
		"state": "pending",
		"time_utc": 1617391800
	},
	"description": "My Wednesday broadcast",
	"job_id": "15981d93877e490185e6ebc9af2c79a3",
	"type": "jobstartstop",
	"workflow_id": "d44cf29f4f184757ac0995beb8a0097e"
}

取消調度程序

您可以使用刪除要求：

https://api.bcovlive.io/v1/scheduler/jobstartstop/{workflow_id}

您也可以在激活後執行此操作，以取消取消激活。

標頭

唯一需要的標頭是：

X-API-KEY: your API Key

要求正文

沒有此請求的請求主體。

樣品回复

 
{
		"account_id": "1752604059001",
		"activation": {
			"state": "cancelled",
			"time_utc": 1617386100
		},
		"deactivation": {
			"state": "cancelled",
			"time_utc": 1617391800
		},
		"description": "My Wednesday broadcast",
		"job_id": "15981d93877e490185e6ebc9af2c79a3",
		"type": "jobstartstop",
		"workflow_id": "d44cf29f4f184757ac0995beb8a0097e"
	}

通知

您可以選擇在創建工作流程時配置通知。您需要為我們的服務提供一個 URL 郵政到。當調度程序進行激活和取消激活調用時，我們將發送通知。您還可以配置接收通知n激活和/或停用前幾秒。見實時調度程序通知更多細節。

樣品索取主體

 
{
		"job_id": "your Job ID",
		"activation": {
			"time_utc": 1612274100,
			"notification": 600
		},
		"deactivation": {
			"time_utc": 1612281600,
			"notification": 600
		},
		"description": "My Tuesday broadcast",
		"notification_url": "http://example.com/live/jobstartstop/callbacks"
	}
