管理實時設置

在本主題中，您將學習如何管理布萊特灣直播的設定。

即時設定可讓您管理填色板和即時認證。

筆記：Brightcove Live 所使用的伺服器端廣告設定是設定為伺服器端廣告設定管理的一部分。有關信息，請參閱配置服務器端廣告設置.

管理填滿板

填色板是指即時串流廣告片段有空白時，可以提供的內容。刻板可以用來提供一個「馬上回來」的消息或任何其他要顯示的內容。所有Zencoder 輸入格式支持。

筆記：填充板也可以使用即時 API 來管理。有關信息，請參閱Brightcove Live API：服務器端廣告插入 (SSAI).

若要新增填色板，請依照下列步驟執行。

  1. 點擊行政 > 實時設置 Studio標頭中的鏈接。
    筆記：您必須是帳戶管理員才能訪問 ADMIN 菜單。
  2. 點擊+添加填充板岩。
  3. 進入一個名稱。
  4. 輸入來源網址為石板。
  5. 點擊保存。石板的攝取過程將開始。

要刪除填充板，請單擊刪除按鈕 () 旁邊的填充石板。

管理認證

即時 API 和即時模組都會使用認證。如果您要建立新的即時視訊雲端認證，則必須先建立 API 驗證認證。建立即時視訊雲端認證時，會使用建立的用戶端識別碼和用戶端密碼。建立 FTP、S3 或 RTMP 認證時，不需要 API 驗證憑證。

如果要創建S3，FTP或RTMP憑據，請跳至第5步。若要建立即時視訊雲端認證，請先建立 API 認證。

  1. 點擊行政 Studio標頭中的鏈接。
    筆記：您必須是帳戶管理員才能訪問 ADMIN 菜單。
  2. 點擊API認證關聯。API 驗證頁面將會開啟，顯示您目前的用戶端註冊。
  3. 點擊註冊新申請創建一個新的API憑據。應該選取下列 API。
    筆記：有關創建 API 身份驗證憑據的信息，請參閱管理 API 身份驗證憑據.
  4. 創建憑據後，保存客戶編號和客戶機密價值觀。建立即時視訊雲端認證時需要這些資料。
  5. 點擊行政 > 實時設置 Studio標頭中的鏈接。
  6. 點擊+添加憑證。
  7. 進入一個標籤。這是將在內部用於引用憑證的名稱。
  8. 選擇一個類型：
    • 的FTP -用於將VOD資產保存到ftp位置（使用API）
    • 實時MP -用於創建RTMP憑證以供編碼器使用
    • S3 -用於將VOD資產保存到Amazon S3帳戶（使用API）
    • 視頻雲 -用於將VOD資產保存到視頻雲帳戶（使用API）
  9. 為了公鑰/用戶名，請根據憑證類型輸入以下內容：
    • 的FTP -輸入FTP站點的用戶名
    • 實時MP -輸入編碼器將用於驗證的用戶名
    • S3 -輸入S3帳戶的用戶名
    • 視頻雲 - 輸入客戶編號為API身份驗證憑證生成
  10. 為了私鑰/密碼，請根據憑證類型輸入以下內容：
    • 的FTP -輸入FTP站點的密碼
    • 實時MP -輸入編碼器用於驗證的密碼
    • S3 -輸入S3帳戶的密碼
    • 視頻雲 - 輸入客戶機密為API身份驗證憑證生成
  11. 點擊保存。

認證會出現在認證清單中。要刪除憑據，請單擊刪除按鈕 () 旁邊的憑據。
