筆記：此功能僅適用於 Professional 和 Enterprise 帳戶。

建立用戶端註冊

API 身份驗證頁面用於創建新的客戶端註冊。建立註冊時，您可以選取服務和要在註冊中啟用的角色。

如要為您的帳戶建立新的客戶註冊：

  1. 點擊行政 Studio標頭中的鏈接。
  2. 點擊API認證關聯。API 驗證頁面將會開啟，顯示您目前的用戶端註冊。
    筆記：只有您登錄的帳戶的客戶註冊才會顯示。
  3. 點擊註冊新申請。
  4. 進入一個名稱和描述用於客戶註冊。
  5. 選取一個或多個帳戶進行授權。
  6. 選取要啟用此註冊的 API 和權限。
    筆記：選取大量的帳戶和 API 可能會造成 HTTP 標頭大小變大且無效。我們的服務器接受的最大標題大小是 8K，這意味著您必須選擇 30 個或更少的帳戶才能應用憑據。出於安全原因，我們建議您將一組憑據的權限範圍限制為所需的最小帳戶和操作集。
  7. 點擊保存。
  8. 一種客戶編號和客戶機密將被退回。
    筆記：複製客戶端機密並將其作為客戶端機密存儲在安全位置將不會再次顯示。

管理用戶端註冊

API 驗證頁面可用來檢視和管理用戶端註冊。點擊行政 Studio標頭中的鏈接，然後單擊API認證。將會顯示您目前的註冊清單。

若要檢視註冊詳細資訊，請按一下名稱。從這裡，您可以：

  • 修改名稱
  • 複製用戶端識別碼
  • 新增授權的帳戶
  • 選取要啟用的其他 API
  • 刪除認證

點擊保存保存您的更改。

請注意，如果為多個帳戶建立認證，使用者只會在擁有這些帳戶的存取權時才會看到該認證。例如，使用者 1 會為帳戶 A & B 建立認證使用者 2 登入，且只有帳戶 A 的存取權，使用者 2 不會看到認證，因為他們沒有帳戶 B 的存取權。

刪除用戶端註冊

要刪除客戶註冊，請單擊刪除圖標（) 旁邊的註冊或編輯註冊並單擊刪除註冊 .刪除註冊將永久禁用它，並且無法將其撤消。

使用 DFP

如果您目前在 DFP 中設定為內容來源的多個 Video Cloud 帳戶，您有兩種選擇：

  1. 您可以為在 DFP 中設定為內容來源的每個視訊雲端帳戶建立唯一的用戶端識別碼和用戶端密碼。例如，如果您有 10 個視訊雲端帳戶，則視訊雲端的 API 驗證設定中會有 10 個名為 DFP 的應用程式。
  2. 您可以為已在 DFP 中設定為內容來源的所有視訊雲端帳戶建立單一用戶端識別碼和用戶端密碼。例如，如果您有 10 個視訊雲端帳戶，則視訊雲端的 API 驗證設定中會有一個名為 DFP 的應用程式。

無論您選擇了 #1 或 #2 選項，對於您在 DFP 中設定的每個內容來源，都必須提供唯一的帳戶 ID。因此，您並不會減少目前在 DFP 中設定的內容來源數目；您只是將讀取權杖取代為用戶端識別碼和用戶端密碼，這些內容來源可以是相同的，也可以是每個內容來源都是唯一的。

選項 #1 需要更多的時間在前面。不過，如果所有內容來源中的用戶端識別碼和用戶端密碼都相同 (與選項 #2 一樣)，則新增內容來源將會成為負擔，因為您必須產生新的用戶端識別碼和用戶端密碼，並更新 DFP 中的所有內容來源。因此，布萊特灣建議選項 #1。
