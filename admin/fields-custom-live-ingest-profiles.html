    自定義實時提取配置文件的字段

    本主題為Live的攝取配置文件中使用的字段提供參考。

    個人資料欄位

    屬性 類型 描述 預設
    account_id 整數 的Video Cloud帳戶ID 無
    name 串 個人資料名稱-在帳戶中必須唯一 無
    display_name 串 用於顯示在Studio管理員頁面中的配置文件的顯示名稱-可能與name，但應具有描述性，以幫助用戶了解配置文件的預期用途 無
    description 串 資料描述 無
    model_version 整數 設置為1使配置文件顯示在實時模塊中 無
    renditions 陣列 轉譯物件的陣列 []
    packages 陣列 DRM 軟件包的數組-目前不支持 DRM，所以這應該始終是一個空數組 []

    渲染字段

    下表顯示了可在請求正文中用於更新實時提取配置文件的字段。

    屬性 類型 描述 預設
    audio_bitrate 整數 kbps 中的音訊位元率 匹配來源
    audio_codec 串 音訊轉碼器- aac | mp3 由格式或配置文件決定-在大多數情況下，aac最好/ td>
    decoder_bitrate_cap 整數 以kbps為單位，通過緩衝區饋送到解碼器的最大比特率 無
    decoder_buffer_size 整數 以kbps為單位，當使用bitrate_cap 無
    forced_keyframe_rate 數 每X幀強制設置一個關鍵幀，但仍允許用於場景切換等其他關鍵幀。 無
    format 串 影片格式- ts，mp4，flv 無
    h264_profile 串 H.264具有三種常用的配置文件：Baseline（最低），Main和High。 [1-5] baseline
    hls_optimized_ts 布林 針對HLS優化的時間段-僅適用於HLS輸出 無
    keyframe_interval 整數 關鍵幀之間的最大幀數 250
    keyframe_rate 浮動 每秒最大關鍵幀 keyframe_interval一百五十
    label 串 圖像再現的圖像類型- poster要么thumbnail 無
    live_stream 布林 這是否將是實時視頻流 無
    live_dvr_sliding_window_duration 整數 流的持續時間（以秒為單位）可用於LiveDVR交付 300
    max_video_bitrate 整數 以kbps為單位，123 = 123kbps-僅適用於h264 無
    max_frame_rate 浮動 限制frame_rate而不是設置它。除非您的帳戶已被支持60 FPS啟用，否則應將其設置為30 無
    media_type 串 video或audio 無
    video_bitrate 整數 以kbps為單位，123 = 123kbps 無
    video_codec 串 例如h264，ts 由格式或配置文件確定
    width 整數 幀寬度（以像素為單位） 無
    height 整數 幀高度（以像素為單位） 無
