自定義實時攝取配置文件的字段

本主題提供用於內嵌即時設定檔的欄位參考。

設定檔欄位

屬性 類型 描述 預設
account_id 整數 這視頻雲帳戶ID 沒有
name 串 設定檔名稱-帳戶中必須是唯一的 沒有
display_name 串 用於顯示在Studio管理員頁面中的配置文件的顯示名稱-可能與name，但應具有描述性，以幫助用戶了解配置文件的預期用途 沒有
description 串 設定檔說明 沒有
model_version 整數 設置為1使配置文件顯示在實時模塊中 沒有
renditions 陣列 轉譯物件的陣列 []
packages 陣列 DRM 軟件包的數組-目前不支持 DRM，所以這應該始終是一個空數組 []

節目欄位

下表顯示可用於更新即時內嵌設定檔的要求主體中的欄位。

屬性 類型 描述 預設

format 串 影片格式- ts，mp4，flv 沒有
h264_profile 串 H.264 有三個常用的配置文件：Baseline（最低），Main， 和High .[1-5] baseline
hls_optimized_ts 布林 針對 HLS 最佳化的時間區段-僅適用於 HLS 輸出 沒有
